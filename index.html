Вниманию пользователей!!!

Порядок установки сертификата можно скачать здесь.

Для корректной работы с защищенным протоколом HTTPS (наличие сообщений "403 - запрещено. Доступ запрещен.") необходимо установить сертификат mfrb.crt в хранилище "Доверенные корневые центры сертификации", в случае использования браузеров: хром, яндекс браузер, internet explorer, опера. Если же используется firefox то помимо этого нужно в самом бразуере импортировать сертификат через настройки, а также добавить сайт в исключения при первой загрузке.

После проведенных действий браузер необходимо перезагрузить.

Вниманию пользователей информационных систем Министерства финансов Республики Бурятия!!!

ПК "Проект-Смарт Про", ПК "Бюджет-СМАРТ Про", ПК "Cвод-СМАРТ" доступны только как архив

В последнее время регуляторы в сфере информационной безопасности (ФСТЭК России, ФСБ России) все чаще обращают внимание на практическую защищенность информационных систем государственных и муниципальных органов власти.
Один из важнейших вопросов - безопасность взаимодействия пользователей систем при их идентификации (аутентификации). Так, использование незащищенного протокола HTTP при работе с информационными системами, несет риск перехвата логинов и паролей пользователей, несанкционированного доступа к информации и компрометации действий пользователей. Под угрозу может быть поставлена доступность и функционирование всех информационной системы в целом.

На основании изложенного Министерство финансов Республики Бурятия предупреждает о переходе с 01.04.2019 года на протокол HTTPS !!!!! Проверяйте внимательно настройки ссылок для успешной работы в системах (см. ниже).

Вниманию пользователей ПК "Проект-Смарт Про"!!!.

Файлы установки и конфигурации ПК "Проект-Смарт Про" можно скачать здесь. Файл конфигурации после установки скопировать в папку с установленной программой.

Для подключения Проект-Смарт Про к базе Минфина через СЕРВЕР ПРИЛОЖЕНИЙ необходимо:
ОСНОВНЫЕ НАСТРОЙКИ:
Вкладка РЕГИСТРАЦИЯ: Сервер БД: kazna-as4
База: dwh2
В поле ПРОФИЛЬ задается любое имя, под которым сохраняются настройки
Вкладка СОЕДИНЕНИЕ: Сервер приложений выбрать "ДА"
в поле СЕРВЕР httpS://budget.govrb.ru/dwh2ws/service.asmx <------ обновление от 01.04.2019
Поставить галочку на: "Использовать сервис авторизации"
Вкладка ОБНОВЛЕНИЕ: Поставить галочку на: "ВЕБ-Сервис" и указать http://budget.govrb.ru/updateservice
